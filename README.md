## About the Project 🥱
<div align="center">
    <br/><img src="https://raw.githubusercontent.com/Devparth7-coder/AI-Resume-Analyzer/main/screenshots/RESUME.png" alt="screenshot" /><br/><br/>
    <p align="justify"> 
      A tool which parses information from a resume using natural language processing and finds keywords, clusters them into sectors based on relevance, 
      and shows recommendations, predictions and analytics to applicants and recruiters.
    </p>
</div>

## Scope 😲
i. Converts resume data into structured tabular format & CSV  
ii. Gives recommendations, predictions and scoring  
iii. Helps users improve their resume  
iv. Helps colleges analyze student resume statistics  
v. Provides analytics on trending roles  
vi. Future improvement based on feedback

## Features 🤦‍♂️
### Client Side
- Parse Resume  
- Extract Skills, Keywords, Basic Info  
- Recommended Skills  
- Predicted Job Role  
- Courses & Certificates  
- Resume Tips  
- Overall Resume Score  
- Video Recommendations  

### Admin Side
- View applicant data in tables  
- Export user CSV  
- View all uploaded resume PDFs  
- Feedback & ratings analytics  
- Pie charts for multiple resume parameters
